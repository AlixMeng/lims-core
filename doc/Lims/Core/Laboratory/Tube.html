Overview

This Action transfers the given fraction and type of aliquot from tube-like asset(s) to tube-like asset(s). It takes an array, which contains transfer elements. An element has a source, target, amount, fraction and type parameter. Source and targets are tube-like assets (a tube or spin column). Amount is an amount of an aliquot to transfer. Fraction is the fraction of an aliquot to transfer. Type is the type of the aliquot.
